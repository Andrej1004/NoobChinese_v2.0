Помощь по добавлению текста

Введите название текста, оригинал и перевод, затем нажмите «готово» для начала работы. Для добавления подсказок выберите левой кнопкой мыши группу слов оригинала и соответствующие им слова перевода и нажмите «сохранить соответствие», затем выберете следующую группу. Для добавления подсказки кликните правой кнопкой мыши на нужное слово оригинала, в контекстном меню введите перевод и, если нужно, транскрипцию затем нажмите «сохранить перевод». Для проверки результата нажмите «тестировать». Если все правильно, кликните «сохранить на сайте». Текст будет добавлен после проверки администратором.
Замечание для языков, не использующих пробелов: Для корректной работы при вводе на китайском или японском разделяйте слова пробелами. Также ставте их после знаков припинания, например после "。"
